Interactive Story Game

This is a fun little DND style interactive story game I built originally in C++ but transferred to JavaScript to embed in the website and make it playable here.

To play, read the questions and answers and input either 1 or 2 depending on the answer you want to choose and hit enter or the next button to submit your chosen answer. The object of the game is to find the hidden treasure, Good luck!
